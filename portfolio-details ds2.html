Cafe Web Traffic Analytics Pipeline

Many businesses struggle to understand how customers interact with their websites, limiting their ability to optimize digital experiences and drive conversions. To address this challenge, a clickstream analytics pipeline was developed on AWS to analyze website behavior for a fictional caf√© chain. Apache web server logs were captured in real time using Amazon CloudWatch, transformed into structured JSON enriched with geolocation data, and stored in Amazon S3 for long-term retention. Insights were extracted using CloudWatch Logs Insights and Amazon Athena, while a CloudWatch Dashboard enabled real-time visualization of key metrics such as menu views, order completions, and regional engagement trends. This project showcases skills in cloud-based data engineering, log transformation, pipeline automation, and end-to-end analytics using AWS services including EC2, CloudWatch, S3, and Athena.
