Brazilian E-Commerce Sales Data Pipeline

This project addresses the common business challenge of integrating scattered on-premise data sources into a centralized cloud platform for analytics and machine learning. Using the Brazilian e-commerce dataset from Olist, I worked with data sourced from both CSV files (via HTTP on GitHub) and SQL tables. The goal was to design an automated and scalable data pipeline on Azure to prepare this raw data for downstream business use cases.

I built a parameterized ETL pipeline using Azure Data Factory to ingest multiple files and load them into the Bronze layer of Azure Data Lake Storage Gen2. The raw data was then processed in Azure Databricks, where I applied data cleaning, filtering, and transformation logic. Enrichment was performed using reference data from MongoDB, and the resulting outputs were saved in Parquet format in the Silver layer. Cleaned data was further loaded into Azure Synapse Analytics for business-level aggregations, creation of custom metrics, and final output to the Gold layer in ADLS Gen2.
