AI-Powered Chatbot for GW Physician Assistant Program Admissions

Developed a proof-of-concept AI chatbot to automate responses to frequent admissions inquiries for the GW Physician Assistant Program using Azure AI Foundry. Admissions documents were ingested into a secure Azure Storage Account and indexed using AI Search to enable semantic retrieval. Custom prompt engineering and Python-based filters were applied to refine user inputs before passing them to a large language model (LLM). The system was deployed via a serverless endpoint and integrated into a web application for testing. This architecture enabled retrieval-augmented generation (RAG), allowing the LLM to generate accurate, document-grounded answers in real time. The solution showcases practical application of Azure AI services, vector indexing, and prompt tuning to streamline high-volume information delivery.
